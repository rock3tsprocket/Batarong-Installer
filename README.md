## Installing
> [!NOTE]
> i like coding

1. Download the KDE Plasma version of Debian.
2. Rename correctly. 
3. Run the installer: ```curl https://batarong.github.io/install.sh | sudo bash```  
4. Select the defaults until the end.
5. You're done.
